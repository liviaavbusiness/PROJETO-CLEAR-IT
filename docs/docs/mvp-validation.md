# Casos de Uso

## Caso 1 - Empresa com risco de segurança

Cliente relata preocupação com vazamento de dados.

Sistema deverá identificar:

* necessidade de proteção de identidade;
* necessidade de gestão de vulnerabilidades;
* necessidade de monitoramento contínuo.

Soluções possíveis:

* firewall corporativo;
* SIEM;
* centro de operações de segurança.

---

## Caso 2 — Empresa com infraestrutura instável

Cliente relata indisponibilidade recorrente.

Sistema deverá identificar:

* necessidade de observabilidade;
* baixa resiliência da infraestrutura;
* necessidade de monitoramento contínuo.

Soluções possíveis:

* centro de operações cognitivas;
* observabilidade;
* automação operacional.

---

## Caso 3 — Empresa em expansão tecnológica

Cliente relata dificuldade em escalar ambiente atual.

Sistema deverá identificar:

* necessidade de modernização;
* limitação de infraestrutura atual;
* necessidade de ambiente híbrido.

Soluções possíveis:

* nuvem híbrida;
* virtualização;
* hiperconvergência.
