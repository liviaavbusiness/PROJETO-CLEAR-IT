# Matriz de Funcionalidades - MVP Copiloto Inteligente de Pré-Vendas

Este documento organiza todas as funcionalidades previstas para o MVP inicial do sistema proposto para a Clear IT.

O objetivo é permitir rastreabilidade do desenvolvimento, acompanhamento das próximas sprints e alinhamento entre produto, negócio e tecnologia.

Cada funcionalidade representa uma capacidade central necessária para que o sistema consiga reproduzir parte do raciocínio consultivo atualmente concentrado nos especialistas da empresa.

# Critério de Conclusão do MVP

O MVP será considerado pronto para validação quando todas as funcionalidades críticas e altas estiverem concluídas.

Condição mínima obrigatória:

- F01 concluído
- F02 concluído
- F04 concluído
- F05 concluído
- F06 concluído

Sem essas funcionalidades o sistema não consegue reproduzir o raciocínio consultivo esperado.

---

# Visão Estratégica do Produto

A construção deste MVP não tem como objetivo automatizar vendas.

Seu propósito central é transformar conhecimento consultivo atualmente concentrado em especialistas da Clear IT em inteligência operacional reutilizável.

O sistema deve reduzir dependência individual, aumentar padronização do discovery e permitir maior escala operacional no processo de pré-vendas.
